ACE LabVIEW Toolkit

May 2019

Thank you for using the ACE LabVIEW add-on and toolkit. ACE provides functionality to the LabVIEW development environment via a Shortcut Menu Plugin (SMP), a Quick Drop Keyboard Shortcut (QDKS), and an API to alphabetize cluster elements on the Front Panel and Block Diagram of VIs.

Installation

This section documents installation instructions for the ACE LabVIEW add-on and toolkit.

  1. Download the vipm package for ACE.
  2. Open it with VI Package Manager (available for free at www.jki.net/vipm) and follow the on-screen instructions.
  3. Restart LabVIEW.
