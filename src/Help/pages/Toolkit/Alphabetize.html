Alphabetize VI

Owning Palette: ACE VIs

Requires: ACE

Alphabetizes all elements in all clusters. If the GObject reference is not to a cluster constant or control, then it is silently skipped. Only clusters will be alphabetized.

Error In describes error conditions that occur before this node runs. This input provides standard error in functionality.
Clusters In is an array of references to clusters that should be alphabetized.
Error Out contains error information. This output provides standard error out functionality.
Clusters Out is an array of references to all clusters that were alphabetized.
Change? is TRUE if any clusters were alphabetized; otherwise, FALSE. This will be FALSE if an error occurred.
