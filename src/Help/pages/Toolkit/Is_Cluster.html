Is Cluster VI

Owning Palette: ACE VIs

Requires: ACE

Checks if the GObject reference is to a cluster constant on the Block Diagram or a cluster control on the Front Panel.

Error In describes error conditions that occur before this node runs. This input provides standard error in functionality.
GObject In is the reference to a GObject in a VI to be checked if it is a cluster.
Error Out contains error information. This output provides standard error out functionality.
Cluster? is TRUE if the GObject is a cluster; otherwise, FALSE. This is TRUE if the GObject is a cluster control, indicator, or constant.
GObject Out is a reference to the GObject that was checked if it was a cluster.
